WINDOWS10下安装nextcloud

操作手顺

WINDOWS10下用ubuntu安装nextcloud服务器端

安装流程

  1. 开启windows系统中的wsl服务，在软件管理中
  2. 在store中下载ubuntu18.0.4软件，安装完成后需设置unix系统名和密码。
  3. sudo apt install apache2 mysql-server php7.2 libapache2-mod-php7.2 php7.2-gd php7.2-json php7.2-mysql php7.2-curl php7.2-mbstring php7.2-intl php-imagick php7.2-xml php7.2-zip php7.2-dom
  4. wget https://download.nextcloud.com/server/releases/nextcloud-18.0.4.zip
  5. sudo apt install unzip unzip nextcloud-18.0.0.zip
  6. sudo mv nextcloud /var/www/html/nextcloud/
  7. sudo mkdir /var/www/html/nextcloud/data
  8. sudo chown -R www-data:www-data /var/www/html/nextcloud/ sudo chmod -R 755 /var/www/html/nextcloud/
  9. sudo nano /etc/apache2/sites-available/nextcloud.conf
    在其中输入代码
    ```

DocumentRoot /var/www/html/nextcloud/

Alias /nextcloud "/var/www/html/nextcloud/"

Options +FollowSymlinks AllowOverride All Require all granted Dav off SetEnv HOME /var/www/html/nextcloud SetEnv HTTP_HOME /var/www/html/nextcloud

ErrorLog ${APACHE_LOG_DIR}/error.log CustomLog ${APACHE_LOG_DIR}/access.log combined

</VirtualHost>

>[来源](https://pastebin.com/9rjXe5rd)按ctrl+O存储，回车命名，ctrl+X为返回到前页面。
10. sudo nano /etc/apache2/apache2.conf  
在最结尾的地方追加两行

AcceptFilter http none AcceptFilter https none

>按ctrl+O存储，回车命名，ctrl+X为返回到前页面。
11. sudo service apache2 start
sudo a2ensite nextcloud.conf
sudo service apache2 reload
sudo apt update
12. sudo a2enmod rewrite
sudo a2enmod headers
sudo a2enmod env
sudo a2enmod dir
sudo a2enmod setenvif
sudo a2enmod mime
sudo service apache2 restart
13. sudo service mysql start
sudo mysql
CREATE DATABASE nextdb;
GRANT ALL ON nextdb.* to 'nextcloudUser'@'localhost' IDENTIFIED BY 'userP@ss123';
FLUSH PRIVILEGES;
exit

### 设置
1. 打开网址：httP://localhost/nextcloud/
2. 设置管理员账号密码，关联数据库。
>账号：nextcloudUser  
密码：userP@ss123  
数据库名：nextdb
3. 外网连接
 sudo nano /var/www/html/nextcloud/config/config.php
