gridstudio在docker下的安装

gridstudio安装手顺

一、gridstudio的用途

Grid Studio 是一款基于 Web 的应用程序，看起来和 Google Sheets 或 Excel 很像。不过，它的杀手锏特性是与 Python 的深度集成。 你可以直接用它查看和编辑表格结构的数据，也可以直接在上面编写和执行 Python 代码。

二、安装环境

需要安装git与docker

  1. 同安装手顺1，安装docker，获取本地访问地址：http://192.168.99.100
  2. 安装git,下载gridstudio程序。 ``` git clone https://github.com/ricklamers/gridstudio

cd gridstudio

docker run --name=gridstudio --rm=false -p 8080:8080 -p 4430:4430 ricklamers/gridstudio:release ```

  1. 浏览器访问网址http://192.168.99.100:8080/

    账号密码均为admin
